# Tech Support Chatbot
**Built by Carramah Quiett**

An AI-powered customer service chatbot built with Claude AI, deployed on Vercel.

## Features
- AI-powered tech support for software, hardware, networking, and coding issues
- Full AI governance compliance framework (GDPR, CCPA, FTC)
- Human escalation system with conversation history handoff
- Markdown response formatting
- Session privacy and legal privilege disclosures

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Vercel Serverless Functions (Node.js)
- AI: Anthropic Claude API
- Deployment: Vercel via GitHub

## Governance Compliance
Built in accordance with AI Governance & Requirements Framework v1.0, including mandatory disclosure, data privacy, content guardrails, and human escalation protocols.
